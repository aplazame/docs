|<t id="table.parameter">Parámetro</t> | <t id="table.type">Tipo</t> | <t id="table.required">Requerido</t> | <t id="table.description">Descripción</t>|
|---------|---------|---------|---------|
:merchantId | hash | <t id="table.yes">Sí</t> | `id` de la tienda con el que hacer las peticiones.

 ### Respuesta

```http
HTTP/1.1 200 OK
Content-Type: application/json
X-Aplazame-Media-Type: aplazame.v1
```
```json
{
  "id": "c994e03e42d24be7bf6e1fcbad3334c0",
  "name": "Summertime",
  "start_date": "2016-06-01T00:00:00",
  "end_date": "2016-08-31T23:59:59",
  "interest": 2500,
  "partial": true,
  "num_instalments": [1, 2, 3, 5, 8],
  "created": "2016-03-10T11:08:57.265971"
}
```

|<t id="table.parameter">Parámetro</t> | <t id="table.type">Tipo</t> | <t id="table.description">Descripción</t>|
|---------|---------|---------|
id | hash | `id` del hostname
name | string | Nombre de la campaña
start_date | [ISO 8601](https://es.wikipedia.org/wiki/ISO_8601) | Fecha de inicio de la campaña
end_date | [ISO 8601](https://es.wikipedia.org/wiki/ISO_8601) | Fecha fin de la campaña
interest | [decimal](../../api-reference#decimals) | Tasa de descuento en los intereses
partial | Boolean | Determina si la campaña aplica a todos los productos
