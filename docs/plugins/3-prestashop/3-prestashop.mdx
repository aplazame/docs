# Plugin para PrestaShop
Aquí encontrarás todo lo referente al plugin de Aplazame para PrestaShop.

:::tip
  Ten a mano la **clave API privada** relativa a tu tienda online, la cual puedes obtener desde tu [panel de control](https://vendors.aplazame.com/) en la sección **Preferencias**.
:::

## Descarga
Descarga el plugin para PrestaShop desde [este enlace](https://aplazame.com/static/sdk/prestashop-latest.zip).

## Instalación

### En PrestaShop 8 y 1.7
- Accede al panel de administración de tu tienda creada en PrestaShop.
- Navega hasta la sección **Improve > Modules** y pulsa en el botón **Subir un módulo**.
- Selecciona el archivo descargado o arrástralo hasta la ventana para instalarlo.

### En PrestaShop 1.6
- Accede al panel de administración de tu tienda creada en PrestaShop.
- Navega hasta la sección **Módulos y Servicios** y pulsa en el botón **Añadir un nuevo módulo**.
- Selecciona el archivo descargado y pulsa en el botón **Subir este módulo**.

## Configuración básica
En los ajustes de configuración del plugin debes realizar lo siguiente:
- Incluye tu **clave API privada**.
- Si quieres realizar una prueba, activa el **Modo de Pruebas (Sandbox)** y realiza una compra de prueba usando nuestros [datos de prueba](../../api/test-environment/test-data/).
:::caution
  Recuerda desactivar el **Modo de Pruebas (Sandbox)** cuando termines las pruebas para dejar el método de pago en modo producción y que se puedan llevar a cabo pedidos reales.
:::
