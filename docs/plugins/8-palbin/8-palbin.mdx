# Integración en Palbin
Aquí encontrarás todo lo necesario para integrar Aplazame en tu tienda Palbin.

:::tip
  Ten a mano las **claves API** relativas a tu tienda online, las cuales puedes obtener desde tu [panel de control](https://vendors.aplazame.com/) en la sección **Preferencias**.
:::

## Configura el método de pago Aplazame
- Accede al panel de administración de tu tienda Palbin.
- Ve a **Configuración > Método de pago > Opciones de pago aplazado** y selecciona la opción de **Aplazame** en el desplegable si aparece disponible en tu país.
- Configura el método de pago con tus **claves API**.
- Si quieres realizar una prueba, activa la casilla **Marca esta opción SOLO para realizar pruebas...** y realiza una compra de prueba usando nuestros [datos de prueba](../../api/test-environment/test-data/).
:::caution
  Recuerda desactivar la casilla **Marca esta opción SOLO para realizar pruebas...** cuando termines las pruebas para dejar el método de pago en modo producción y que se puedan llevar a cabo pedidos reales.
:::
