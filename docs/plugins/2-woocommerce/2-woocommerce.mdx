# Plugin para WooCommerce
Aquí encontrarás todo lo referente al plugin de Aplazame para WooCommerce.

:::tip
  Ten a mano la **clave API privada** relativa a tu tienda online, la cual puedes obtener desde tu [panel de control](https://vendors.aplazame.com/) en la sección **Preferencias**.
:::

## Descarga e instalación

### Desde el panel de administración de WordPress
- Accede al panel de administración de WordPress en el que tengas tu tienda gestionada en WooCoomerce.
- Navega hasta la sección **Plugins > Añadir nuevo** y teclea `aplazame` en el buscador de plugins.
- Pulsa en el botón **Instalar ahora** de nuestro plugin para iniciar su descarga e instalación.

### Desde la web de WordPress
- Navega a la página de [nuestro plugin](https://wordpress.org/plugins/aplazame/) en la web de WordPress.
- Descárgalo pulsando en **Download**.
- Accede al panel de administración de WordPress en el que tengas tu tienda gestionada en WooCoomerce.
- Navega hasta la sección **Plugins > Añadir nuevo** y haz click en **Subir plugin**.
- Selecciona el archivo descargado y pulsa en el botón **Instalar ahora**.

## Configuración básica
En los ajustes de configuración del plugin debes realizar lo siguiente:
- Incluye tu **clave API privada**.
- Activa **Aplazame** como método de pago.
- Si quieres realizar una prueba, activa el **Modo de pruebas (Sandbox)** y realiza una compra de prueba usando nuestros [datos de prueba](../../api/test-environment/test-data/).
:::caution
  Recuerda desactivar el **Modo de pruebas (Sandbox)** cuando termines las pruebas para dejar el método de pago en modo producción y que se puedan llevar a cabo pedidos reales.
:::
