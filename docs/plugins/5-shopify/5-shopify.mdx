# App para Shopify
Aquí encontrarás todo lo referente a la app de Aplazame para Shopify.

:::tip
  Ten a mano la **clave API privada** relativa a tu tienda online, la cual puedes obtener desde tu [panel de control](https://vendors.aplazame.com/) en la sección **Preferencias**.
:::

## Vinculación de la app
Agrega la app de Aplazame desde [Shopify App Store](https://apps.shopify.com/aplazame-public-app?locale=es).

## Configuración y activación
Una vez vinculada la app con la tienda, se pedirá incluir la **clave API privada**.
- Pega la **clave API privada** en la configuración de la app y pulsa en el botón **Configurar**.
:::info
  Si la **clave API privada** es válida, la app redirigirá a la sección para activar el método de pago.
:::
- Pulsa en el botón **Activar Aplazame** y ya tendrías Aplazame como método de pago en tu tienda Shopify.
- Si quieres realizar una prueba, activa **Enable test mode** y realiza una compra de prueba usando nuestros [datos de prueba](../../api/test-environment/test-data/).
:::caution
  Recuerda desactivar **Enable test mode** cuando termines las pruebas para dejar el método de pago en modo producción y que se puedan llevar a cabo pedidos reales.
:::
