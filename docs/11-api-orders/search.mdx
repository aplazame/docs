Para buscar en el conjunto de pedidos.

`GET https://api.aplazame.com/orders?q=param1,param2,...`

La API de Aplazame permite una lista de nombres de campos que se buscarán cada vez que se envíe una consulta de búsqueda con el parámetro `q`.

Las búsquedas utilizarán coincidencias parciales insensibles a mayúsculas y minúsculas. El parámetro de búsqueda puede contener varios términos de búsqueda, que deben estar separados por espacios en blanco y / o comas. Si se usan múltiples términos de búsqueda, entonces los objetos serán devueltos en la lista únicamente si todos los términos indicados coinciden
