# API de campañas
La *API de Campañas* permite gestionar las campañas de financiación del comercio.

Consulta la documentación sobre las [campañas de financiación](https://ayuda.aplazame.com/hc/es-es/sections/360004277640-Campa%C3%B1as-de-financiaci%C3%B3n) como estrategia para incrementar las ventas de tu comercio.
