# Proceso de checkout
Este tutorial te guiará en la integración de Aplazame como método de pago en tu comercio online usando una API REST JSON y JavaScript, o SDKs nativos para plataformas móviles.

Si dispones de alguna plataforma de e-commerce consulta la documentación específica de [plugins](../../plugins/intro/).

:::caution
Antes de comenzar la integración con nuestro método de pago, te recomendamos consultar nuestra sección de [primeros pasos](https://ayuda.aplazame.com/hc/es-es/articles/360014854679--C%C3%B3mo-me-creo-una-cuenta-como-tienda-en-Aplazame-) para completar el proceso de alta y obtener tus credenciales.
:::
