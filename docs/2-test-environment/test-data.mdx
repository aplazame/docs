# Datos de prueba
Puedes utilizar los siguientes datos para simular diferentes estados de la solicitud de financiación en el modo de pruebas.

Utiliza la siguiente tarjeta para validar el método de pago de la solicitud de financiación:

Campo |Valor | Caducidad | CVV
|----- | ----- | -------- | ---
Tarjeta | 4111 1111 1111 1111 | 11/25 | 123

<br/>

### España

Utiliza los siguientes documentos para obtener distintos resultados a la solicitud de financiación:

Campo | Valor | Resultado | Descripción
------| ----- | --------- | ---------
DNI | 99999998 T | KO | No supera los criterios de admisión de Aplazame
DNI | 99999995 C | Pending | Debe completar el reto de validación de identidad
DNI | Otra combinación válida | OK | El crédito es aceptado
