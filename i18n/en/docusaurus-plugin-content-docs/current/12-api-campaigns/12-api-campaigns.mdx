# Campaigns API

The *API Campaigns* allow you to manage the financing campaigns of your store.

Consult the documentation on the [financing campaigns](https://ayuda.aplazame.com/hc/es-es/sections/360004277640-Campa%C3%B1as-de-financiaci%C3%B3n) as a strategy to increase sales in your store.
